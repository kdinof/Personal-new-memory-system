# Personal Memory System

Система персональной продуктивности на базе Claude Code с постоянной памятью, навыками (skills) и еженедельными ретроспективами.

## Зачем

Claude Code не помнит контекст между сессиями. Этот проект решает проблему: Claude читает файлы памяти при старте сессии и знает, кто вы, над чем работаете и какие паттерны уже выявлены.

## Как это работает

1. **Первый запуск** — скилл `bootstrap` через диалог узнает о вас и настраивает систему
2. **Каждая сессия** — Claude читает файлы из `memory/` и понимает контекст
3. **Ежедневно** — `daily-retro` помогает зафиксировать результаты дня
4. **Еженедельно** — `weekly-reflection` анализирует неделю, `weekly-planning` планирует следующую
5. **Задачи** — `task-manager` собирает задачи из ретро и OKR в единый файл

## Структура проекта

```
├── CLAUDE.md                 # Главный файл инструкций для Claude
├── memory/                   # Долгосрочная память
│   ├── USER.md              # Кто пользователь
│   ├── SOUL.md              # Личность и тон бота
│   ├── learnings.md         # Паттерны из ретроспектив
│   ├── life-projects.md     # Активные проекты
│   └── current-focus.md     # Фокус текущей недели
├── retros/                   # Ретроспективы
│   └── YYYY/WXX/            # По годам и неделям
├── skills/                   # Навыки Claude Code
│   ├── bootstrap/           # Первоначальная настройка
│   ├── daily-retro/         # Ежедневная ретроспектива
│   ├── weekly-reflection/   # Анализ недели
│   ├── weekly-planning/     # Планирование недели
│   └── task-manager/        # Управление задачами
└── hooks/                    # Хуки сессий
    └── inject-dates.sh      # Инъекция текущей даты
```

## Навыки (Skills)

| Навык | Что делает | Как вызвать |
|-------|-----------|-------------|
| **bootstrap** | Настраивает систему через диалог: узнает о вас, задает тон бота | Автоматически при первом запуске |
| **daily-retro** | Структурированная вечерняя ретроспектива: One Win, Main Goal, Brain Dump | "Let's do a retro", "daily retro" |
| **weekly-reflection** | Анализ недели по 6 секциям + рекомендации + обновление learnings | "Analyze my week" |
| **weekly-planning** | Планирование недели по Drucker (Analyze/Eliminate/Time Block) | "Plan my week" |
| **task-manager** | Управление задачами в Obsidian Tasks формате + SMART + авто-синк | "Add task", "sync tasks" |

## Система памяти

Память организована в 5 файлов с разной частотой обновления:

- **USER.md** — кто вы, ваши ценности, стиль работы (обновляется редко)
- **SOUL.md** — как бот должен общаться, тон, правила (настраивается один раз)
- **current-focus.md** — приоритеты текущей недели (обновляется еженедельно)
- **life-projects.md** — активные проекты со статусами (обновляется еженедельно)
- **learnings.md** — паттерны и инсайты из ретроспектив (накапливается)

## Быстрый старт

1. Склонируйте репозиторий
2. Откройте папку в Claude Code: `claude` в корне проекта
3. При первом запуске автоматически запустится `bootstrap` — ответьте на вопросы
4. Система готова к работе

## Совместимость с Obsidian

Файл `tasks.md` использует формат [Obsidian Tasks](https://publish.obsidian.md/tasks/) с emoji-синтаксисом. Если подключить проект к Obsidian и включить плагин Tasks, задачи будут отображаться с приоритетами, датами и рекуррентностью.

## Требования
